# dotfiles

## Init

Set up a new environment or update an environment by running the `init.sh` script using one of the following methods:

### Option 1: Curl
```bash
curl -fsSL https://raw.githubusercontent.com/anuragashok/dotfiles/refs/heads/main/init.sh | bash
```


### Option 2: Wget
```bash
wget -qO- https://raw.githubusercontent.com/anuragashok/dotfiles/refs/heads/main/init.sh | bash
```
